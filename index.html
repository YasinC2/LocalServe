📖 How to Use LocalServe

Getting Started

  • Tap "📂 Open" to select a local folder containing your web project files.
  • The app will automatically find and load index.html or default.html.
  • All relative and absolute paths within your HTML files will resolve correctly.

File Explorer

  • Browse all files and folders inside your project.
  • Tap any HTML file to load it in the Player.
  • Navigate using the breadcrumb trail at the top.

Favorites

  • Tap the button to save the current folder as a favorite.
  • Access favorites from the File Explorer tab.
  • Quickly switch between different projects.

Auto-hide Bars

  • In Settings, enable auto-hide to get a full-screen view in the Player tab.
  • Choose single, double, or triple click to toggle the bars.
  • You can also reveal bars by touching near the top or bottom edge of the screen.

JavaScript Toggle

  • Disable JavaScript in Settings to sandbox loaded pages without script execution.
  • Useful for previewing static content safely.

Offline Use

  • This app works fully offline once installed.
  • On Android, use Chrome's "Install app" or "Add to Home Screen" option.
  • The previously opened folder will be restored on next launch (permission required).

Requirements

  • Chromium-based browser (Chrome 86+, Edge 86+).
  • File System Access API support.
  • For Android: Chrome 86+ with the API flag enabled, or newer versions with native support.
